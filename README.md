# Solid Reads

Solid Reads is a Ruby Sinatra web app for posting book reviews.

## Installation

To make sure you have all the right Ruby gems run:

```bash
bundle install
```
before starting the server.

## Usage
 
```bash
shotgun
```
runs the server to get started
